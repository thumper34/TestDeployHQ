AnglWP - AngularJS WordPress Application

AnglWP is a web application that integrates WordPress content within outside websites. Currently a work in progress this application allows outside websites/apps to pull individual posts, categories and custom post types from another WordPress install. This is an ideal use of AngularJS to add webpage content to IOS/Droid apps.
